Getting Started

System Overview

DLMs-Pro is a comprehensive professional loan management system designed for microfinance institutions. It provides complete borrower lifecycle management, loan origination, approval workflows, repayment tracking, and advanced reporting capabilities.

Quick Start Guide

  1. Login with your assigned credentials (username, password, role, branch)
  2. Navigate using the sidebar menu to different modules
  3. Start by adding borrowers through the Borrowers section
  4. Create loan applications for approved borrowers
  5. Process loan approvals through the Approval workflow
  6. Record repayments as they are received
  7. Generate daily reports and analytics

Keyboard Shortcuts

  • Ctrl+K — Quick search (in Help section)
  • Esc — Close modals/menus

Managing Borrowers

Adding a New Borrower

  1. Navigate to the Borrowers section
  2. Click "Add Borrower" button or use the FAB (+) button
  3. Fill in personal information (name, contact, ID)
  4. Add financial information (occupation, income, credit score)
  5. Enter guarantor details
  6. Optionally upload a photo
  7. Click "Save Borrower"

Searching & Filtering

Use the advanced search feature to find borrowers by name, ID number, phone number, status, or branch. Click "Advanced Options" to reveal additional filters.

Loan Management

Creating a New Loan

  1. Navigate to Loans section
  2. Click "New Loan" button
  3. Select borrower from dropdown (only active borrowers shown)
  4. Enter principal amount, interest rate, and term
  5. Select interest type (flat or reducing balance)
  6. Set purpose, collateral, and notes
  7. Choose status (pending approval or auto-approved)
  8. Submit the loan application

Repayments

Recording Repayments

  1. Navigate to Repayments section
  2. Click "Record Payment" button
  3. Select the active loan from dropdown
  4. Enter payment amount
  5. Choose payment method (Cash, M-Pesa, Bank Transfer, etc.)
  6. Add reference number if available
  7. Confirm the transaction

Reports & Analytics

Daily Financial Report

Generate comprehensive daily financial reports showing opening cash balance, total loans disbursed, total repayments collected, processing fees, other income sources, penalties collected, operating expenses, and closing balance calculation.

Admin Panel Guide

User Management

The admin panel provides complete control over system users including creating new user accounts with specific roles, editing existing user permissions, deactivating or suspending users, viewing user activity logs, and managing role-based access control matrix.

Settings

General Settings

Configure organization name, currency defaults, date formats, and basic application preferences.

Loan Defaults

Set default interest rates, minimum/maximum loan amounts, default terms, and interest calculation methods.
